## ATM System (Core Java based)

This is a Console based ATM System developed using core java.
Its demonstrates basic ATM functionalities like-Withdraw money,
deposit money, Check balance, Enter new custom otp.

# Features
- Check balance
- deposit money
- Withdraw money
- Update OTP
- Convert total INR to USD

# Technology used
- core java
- Inner class like-Member inner class, static inner class,
  method local inner class,Anonymous inner class.
- OOP concepts

# How to Run
1. Compile the program:
   javac ATMSystem.java
2. Run the program:
   java ATMSystem
